This section explains workflows in the system.
Booking Appointment: Select lecturer → Choose time → Check availability → Confirm.
Login: Enter details → Validate → Access or Retry.
Attendance: Input student → Select status → Save.
Approval: View request → Approve/Reject.
Report: Select → Fetch → Generate → Display.
View Attendance: Request → Fetch → Display.
Notification: Trigger → Send Email & Update System (parallel).
Maintenance: Check → Update → Restart.
