This section describes the lifecycle of key system objects using state diagrams.
Appointment: Requested → Approved/Rejected → Completed/Cancelled.
Attendance: NotRecorded → Recorded → Updated.
User Account: Registered → Active → Suspended → Active.
Booking Slot: Available → Reserved → Booked → Completed.
Report: Generated → Viewed → Archive
Notification: Created → Sent → Read.
Session: LoggedOut → LoggedIn → LoggedOut.
Request: Pending → Approved/Rejected → Closed
